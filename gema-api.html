2. ¿Cómo generar el token externamente?

Para obtener un token válido, se debe consumir el siguiente servicio técnico:

URL del servicio https://www.banxico.org.mx/GEMA/API/generaTokenAPIIF
Método POST
Content-Type application/json
Formato de respuesta Texto (String)
Datos de entrada No se requieren datos de entrada

Servicio /consultarInformacion

Este servicio permite consultar información de diversas fuentes de base de datos de manera dinámica y flexible.

Detalles del servicio

URL del servicio https://www.banxico.org.mx/GEMA/API/consultarInformacion
Método POST
Content-Type application/json
Formato de respuesta JSON (Objeto con arreglo de datos)

Descripción de entrada (Body)

Envía un objeto JSON con los siguientes parámetros:

Campo Tipo Descripción Requerido
token Texto Código alfanumérico generado en el servicio de autenticación. Obligatorio
fuente Texto ID de la fuente (ej: ID_FUENTE). Obligatorio
camposVisibles Texto Nombres de campos separados por comas que se desean obtener. Obligatorio
campos Texto Campos a los cuales se les va a aplicar un filtro. Opcional
operadores Texto Operadores lógicos para los filtros (=, >, <, LIKE). Opcional
valores Texto Valores correspondientes a los filtros definidos. Opcional
camposAgrupador Texto Campos para aplicar operaciones de agregación (SUM, AVG). Opcional
operadoresAgrupador Texto Funciones de agrupación a aplicar. Opcional
agruparPorCampos Texto Campos base para la agrupación (GROUP BY). Opcional
ordenamiento Texto Campos de orden. Usa 1 (Asc) o -1 (Desc). Opcional
limite Texto Número de registros. 0 para traer todos. Opcional
paginado Texto 1 para activar paginación, 0 para desactivar. Opcional
pagInicio Texto Número de página inicial (Solo si paginado es 1). Condicional
pagFin Número Número de página final (Solo si paginado es 1). Condicional
